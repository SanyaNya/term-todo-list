# term-todo-list

## Build: 
 
Linux:
 * cmake -DCMAKE_BUILD_TYPE=[Build, Release] -S . -B build
 * cmake --build build
 * cmake --install build

Windows:
 * cmake -S . -B build
 * cmake --build build --config [Debug, Release]
 * cmake --install build

## Note: don't forget use \\ for shell special symbol(e.g. *, <, >, ")
